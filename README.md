# 🎓 Öğrenci Not Hesaplama Sistemi

Bu proje, C programlama dili kullanılarak geliştirilmiş, öğrencilerin sınav notlarına göre harf notlarını hesaplayan temel bir konsol uygulamasıdır. 

Algoritma kurma ve problem çözme yeteneklerimi geliştirmek amacıyla hazırladığım bu projede, kullanıcı deneyimini ve veri doğruluğunu ön planda tuttum.

## 🚀 Projenin Özellikleri

* **Veri Doğrulama (Input Validation):** Kullanıcının 0-100 aralığı dışında geçersiz bir not girmesi durumunda sistem uyarı verir ve hatalı işlemi engeller.
* **Koşullu İfadeler:** `if-else if` yapıları kullanılarak not aralıklarına göre (AA, BB, CC, vb.) hassas harf notu belirlemesi yapılır.
* **Kullanıcı Dostu Arayüz:** Konsol ekranında net ve anlaşılır yönlendirmeler bulunur.

## 💻 Kullanılan Teknolojiler
* **Dil:** C Programming Language
* **Kavramlar:** Temel I/O (stdio.h), Değişken Yönetimi, Karar Yapıları (Conditionals)
